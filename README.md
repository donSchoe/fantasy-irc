# fantasy-irc

A modular, event-driven IRC client/bot Ruby gem with plugin support

## Installation

Add this line to your application's Gemfile:

    gem 'fantasy-irc'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fantasy-irc
